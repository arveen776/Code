# Handwriting Engine MVP

A simple handwriting engine that converts text into handwritten-looking images using random character variations.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Prepare your glyph images:
   - Draw each character 5 times on paper (black ink on white)
   - Scan or photograph
   - Crop each character into individual images
   - Save them in `glyph_raw/` using the naming convention: `CHAR_VARIATION.png`
     - Example: `a_1.png`, `a_2.png`, `a_3.png`, `a_4.png`, `a_5.png`
     - For special characters, use aliases: `space_1.png`, `quote_1.png`, `backslash_1.png`, etc.

3. Preprocess the glyphs:
```bash
cd src
python preprocess_glyphs.py
```

4. Generate handwritten text:
```bash
cd src
python main.py
```

The output will be saved in `output/example.png`.

## File Naming Convention

- Normal characters: `CHAR_VARIATION.png` (e.g., `a_1.png`, `a_2.png`, `a_3.png`, `a_4.png`, `a_5.png`)
- Special characters use aliases:
  - `space_1.png` → space character
  - `quote_1.png` → " (double quote)
  - `backslash_1.png` → \ (backslash)
  - `colon_1.png` → : (colon)
  - `semicolon_1.png` → ; (semicolon)

## Project Structure

```
handwriting_engine/
  glyph_raw/        # Raw character images (you add these)
  glyphs/           # Processed glyphs (generated by preprocess_glyphs.py)
  src/
    config.py       # Configuration (charset, paths)
    preprocess_glyphs.py  # Clean and process raw images
    glyph_loader.py       # Load processed glyphs into memory
    renderer.py          # Render text to image
    main.py              # Main entry point
  output/           # Generated handwritten images
```
