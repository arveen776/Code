Place your raw character images here.

Naming convention:
- Normal characters: CHAR_VARIATION.png
  Examples: a_1.png, a_2.png, a_3.png, a_4.png
            b_1.png, b_2.png, b_3.png, b_4.png
            1_1.png, 1_2.png, 1_3.png, 1_4.png

- Special characters (use aliases):
  space_1.png, space_2.png, space_3.png, space_4.png → " " (space)
  quote_1.png, quote_2.png, quote_3.png, quote_4.png → " (double quote)
  backslash_1.png, backslash_2.png, backslash_3.png, backslash_4.png → \ (backslash)
  colon_1.png, colon_2.png, colon_3.png, colon_4.png → : (colon)
  semicolon_1.png, semicolon_2.png, semicolon_3.png, semicolon_4.png → ; (semicolon)

You need 4 variations of each character you want to support.
Supported formats: .png, .jpg, .jpeg
