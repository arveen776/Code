# Cool Animation Project

A simple and cool animated web page with interactive particles and gradient backgrounds.

## Quick Start

### Option 1: Python Server (Recommended)
```bash
python server.py
```
The server will start on `http://localhost:8000` and automatically open in your browser.

### Option 2: Node.js Server
```bash
npm install
npm start
```

### Option 3: Direct Open
Simply open `index.html` in your browser (though some features work better with a server).
